# PyAlert-SIEM

Mini SIEM desenvolvido em Python para estudo de:
- análise de logs
- monitoramento de eventos
- fundamentos de SOC
- detecção de atividades suspeitas

## Funcionalidades
- leitura de logs
- detecção de eventos suspeitos
- geração de alertas simples

## Tecnologias
- Python

## Objetivo
Praticar conceitos utilizados em ambientes SOC e Blue Team.
